# Player Workload Alert Dashboard

## Project Overview
This project proposes a simple analytics dashboard that tracks player workload over time. The idea is to help a sports organization identify when a player may be carrying too much physical load from training and matches. The dashboard would present workload trends in a clear way so decision makers can quickly see which players may need rest, reduced training or closer monitoring.

## Decision-Making Problem
Sports organizations often need to decide when a player should continue at the same workload, when training should be reduced or when extra recovery may be needed. Without a clear tool, these decisions may depend too much on observation alone. This project is intended to support decisions related to player rotation, recovery planning and training adjustments.

## Proposed Analytics Approach
The project would use data such as minutes played, number of matches in a short period, training load and recovery information. The analysis would compare recent workload against a player’s normal level and highlight unusual increases. The output would be a simple dashboard that shows trends, warning flags and workload summaries for each player. 

## Use by Decision Makers
Coaches, front office executives and team managers would use this tool before matches and training sessions. They would review the dashboard to identify players who may be at risk of fatigue or overload. The tool would support practical decisions such as resting a player, reducing training intensity, or changing player rotation plans.

## Connection to Chapter 7
This project is currently in the creative phase of innovation. The main idea has been identified and its potential value to the organization is clear. However, it has not yet been prototyped, tested with decision makers or integrated into the team’s daily workflow.
